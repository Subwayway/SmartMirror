# SmartMirror
  - 카메라 영상인식을 통해 사용자를 구분하여 근태관리 및 개인 생활관련 정보를 거울 형태로 제공하여
  미관성과 실용성을 갖춤.
## RPI
  - 영상인식
    - PI camera
    - OpenCV
    
  - 근태관리 DB
    - 구글 시트 API:https://developers.google.com/sheets
    
  - 뉴스,날씨 정보
    - Bs4 크롤링
    
  - 교통 검색
    - Odsay API:https://lab.odsay.com/
